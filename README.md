# DayFlow v3.3 — Decision Architecture

Built on Stephen Covey's 7 Habits of Highly Effective People.

**[→ Open App](https://filipefa1998-sudo.github.io/dayflow-planner/)**

**[→ View Presentation](https://github.com/filipefa1998-sudo/dayflow-planner/raw/main/DayFlow-Presentation.pptx)**

---

DayFlow is a decision architecture, not a task manager. It enforces Covey's framework through hard constraints — max 3 daily priorities, max 4 roles, every priority connected to a role — and measures behavioral alignment through an Effectiveness Score across four dimensions.

Built with Firebase Firestore, Gemini AI, and a single HTML file deployed on GitHub Pages.
